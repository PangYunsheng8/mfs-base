### 问答题
##### 1. 什么是 Canvas？Canvas 能干那些事？
\<canvas> 元素用于图形的绘制，通过JavaScript来完成。 canvas 可以绘制路径,盒、圆、字符以及添加图像。
##### 2. 画笔颜色，填充颜色各如何设置？
fillStyle和strokeStyle。
##### 3. Canvas 中如何画圆？如何填充圆？
画圆可以用arc方法，用fill方法填充圆。
##### 4. Canvas 如何改变坐标系的位置和单位长度？
用translate方法和scale方法。
